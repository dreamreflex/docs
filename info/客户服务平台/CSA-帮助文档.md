## 系统简介

### 什么是云梦镜像客户服务系统？

**云梦镜像客户服务系统（Customer Service Application，简称 CSA）** 是一个专为客户设计的在线服务平台。通过这个系统，您可以：

- **提交和跟踪工单**：遇到任何问题（财务、技术、业务）都可以提交工单，我们的团队会及时回复
- **管理业务账号**：维护您的公司信息、税号、联系方式等对公资料
- **管理合同**：查看合同文档，管理合同的签署状态
- **查看项目信息**：了解项目进度、下载项目文件、查看相关链接和任务
- **管理个人资料**：更新您的个人信息和密码

### 系统特点

**安全可靠**：所有数据都经过加密保护，只有您本人可以查看自己的信息  
**操作简单**：界面清晰直观，无需技术背景即可使用  
**随时可用**：支持电脑和手机浏览器访问，随时随地处理业务  
**响应及时**：工单系统确保您的问题得到及时处理

## 快速开始

### 第一步：注册账号

1. 打开系统登录页面
2. 点击"注册"按钮
3. 输入您的邮箱地址
4. 点击"发送验证码"，系统会向您的邮箱发送6位数字验证码
5. 输入收到的验证码、设置密码（至少8位）
6. 完成注册

**提示**：
- 验证码有效期为10分钟
- 如果未收到邮件，请检查垃圾邮件文件夹
- 开发环境下，验证码也会显示在服务器日志中（仅限开发测试）

### 第二步：登录系统

1. 在登录页面输入您的邮箱和密码
2. 点击"登录"
3. 登录成功后，您将进入系统首页（仪表盘）

**提示**：
- 登录状态会保持30分钟，超时后需要重新登录
- 建议使用常用浏览器，并允许保存登录状态

### 第三步：探索功能

登录后，您会看到左侧导航菜单，包含以下功能模块：

- **仪表盘**：系统概览和快速操作
- **个人资料**：管理个人信息
- **业务账号**：管理对公业务信息
- **我的合同**：查看合同文档，管理签署状态
- **工单管理**：提交和跟踪问题
- **我的项目**：查看项目相关信息
- **OAuth 应用**：管理第三方应用授权

## 功能详解

### 1. 仪表盘

**位置**：登录后首页

**功能说明**：
仪表盘是您进入系统后看到的第一个页面，这里提供了：

- **欢迎信息**：显示您的邮箱和账号状态
- **快速操作**：快速创建工单、查看工单等常用功能
- **账号信息**：显示您的邮箱、角色、账号状态和注册时间

**使用建议**：
- 定期查看仪表盘，了解系统最新动态
- 使用快速操作按钮，提高工作效率

### 2. 个人资料

**位置**：左侧菜单 → "个人资料"

**功能说明**：
在这里您可以管理自己的个人信息：

#### 基本资料

- **姓名**：您的真实姓名（可选）
- **联系方式**：您的手机号码或固定电话（可选）
- **头像链接**：头像图片的网址（可选）
- **个人简介**：简单的自我介绍（可选）

**操作步骤**：
1. 进入"个人资料"页面
2. 填写或修改需要更新的信息
3. 点击"保存资料"按钮
4. 系统会提示"资料已更新"

**注意事项**：
- 所有字段都是可选的，您可以根据需要填写
- 这些信息会显示给系统管理员，有助于更好地为您服务
- 不填写任何信息也不会影响系统使用

#### 修改密码

**操作步骤**：
1. 在"个人资料"页面找到"修改密码"部分
2. 输入当前密码
3. 输入新密码（至少8位）
4. 再次确认新密码
5. 点击"修改密码"按钮

**安全提示**：
- 建议定期更换密码
- 密码应包含字母、数字，避免使用过于简单的密码
- 如果忘记密码，请联系系统管理员

### 3. 业务账号

**位置**：左侧菜单 → "业务账号"

**功能说明**：
业务账号用于管理您的对公业务信息，包括公司抬头、税号、联系方式等。这些信息将用于：
- 开具发票
- 邮寄对账单和票据
- 进行业务通知

#### 申请业务账号

如果您还没有业务账号：

1. 进入"业务账号"页面
2. 点击"申请创建业务账号"按钮
3. 系统会创建一条待审核的记录
4. 等待管理员审核并分配专属邮箱

**账号状态说明**：

- **待审核**（橙色标签）：您已提交申请，等待管理员处理
- **已启用**（绿色标签）：账号已激活，可以使用专属邮箱
- **已禁用**（红色标签）：账号被禁用，请联系管理员

#### 管理业务资料

业务账号创建后，您可以维护以下信息：

- **公司抬头**：公司的正式名称
- **税号**：税务登记号
- **认证信息**：如一般纳税人资格等
- **通信地址**：用于接收对账单、票据等的邮寄地址
- **联系人姓名**：委托人或联系人姓名
- **联系方式**：联系人电话或邮箱

**操作步骤**：
1. 进入"业务账号"页面
2. 在"业务资料"部分填写或修改信息
3. 点击"保存业务资料"按钮

**注意事项**：
- 专属邮箱和用户识别码由管理员分配，您无法自行修改
- 建议填写完整信息，以便顺利开具发票和邮寄文件
- 信息更新后，管理员可以看到最新版本

### 4. 我的合同

**位置**：左侧菜单 → "我的合同"

**功能说明**：
在这里您可以查看所有与您相关的合同文档，并管理合同的签署状态。

#### 查看合同列表

**操作步骤**：
1. 进入"我的合同"页面
2. 系统会显示合同列表，包括：
   - **合同标题**：合同名称
   - **签署状态**：
     - **待签署**（橙色标签）：合同尚未签署
     - **已签署**（绿色标签）：合同已完成签署
   - **创建时间**：合同创建时间

#### 管理签署状态

您可以通过开关组件快速切换合同的签署状态：

**操作步骤**：
1. 在合同列表中，找到需要更新状态的合同
2. 使用"签署状态"开关：
   - 开关显示"待签署"时，表示合同未签署
   - 开关显示"已签署"时，表示合同已签署
3. 点击开关即可切换状态
4. 系统会自动保存更改，并显示成功提示

**状态说明**：
- **待签署**：合同尚未完成签署流程
- **已签署**：合同已完成签署，可以用于后续业务

**注意事项**：
- 签署状态会实时保存，无需额外操作
- 状态更改后，管理员可以看到最新状态
- 建议在完成实际签署后及时更新状态

#### 查看合同详情

**操作步骤**：
1. 在合同列表中，点击"查看"按钮
2. 系统会弹出合同详情窗口，显示：
   - 合同标题
   - 合同内容（以 Markdown 格式显示，支持标题、列表、链接等格式）
3. 关闭窗口即可返回列表

**注意事项**：
- 合同内容为只读，您无法修改
- 如需修改合同内容，请联系管理员
- 建议定期查看合同，了解最新条款

### 5. 工单管理

**位置**：左侧菜单 → "工单管理"

**功能说明**：
工单系统是您与我们的主要沟通渠道。您可以提交各种问题，我们的团队会及时回复。

#### 工单类型

系统支持三种类型的工单：

- **财务问题**（绿色标签）：账单、付款、发票等相关问题
- **技术问题**（蓝色标签）：系统bug、功能异常、登录问题等
- **业务问题**（橙色标签）：业务咨询、流程问题等

#### 创建工单

**操作步骤**：
1. 进入"工单管理"页面
2. 点击右上角"创建工单"按钮
3. 填写工单信息：
   - **工单标题**：简要描述问题（必填，最多200字）
   - **工单类型**：选择问题类型（必填）
   - **问题描述**：详细描述您遇到的问题（必填）
4. 点击"确定"提交工单

**提示**：
- 描述越详细，我们越能快速解决问题
- 可以附上截图、错误信息等有助于诊断问题的内容

#### 查看工单列表

工单列表显示所有您创建的工单，包括：

- **标题**：工单标题
- **类型**：工单类型标签
- **状态**：
  - **待回复**（橙色标签）：已提交，等待管理员处理
  - **已回复**（绿色标签）：管理员已回复
- **创建时间**：工单提交时间

**筛选功能**：
- 使用"状态筛选"下拉框，可以只查看"待回复"或"已回复"的工单
- 点击"刷新"按钮，更新工单列表

#### 查看工单详情

**操作步骤**：
1. 在工单列表中，点击"查看"按钮
2. 系统会显示工单的完整信息：
   - 标题和类型
   - 问题描述
   - 状态
   - 创建时间
   - 如果已回复，会显示回复内容和回复时间

#### 编辑工单

**操作步骤**：
1. 在工单列表中，找到状态为"待回复"的工单
2. 点击"编辑"按钮
3. 修改标题或问题描述
4. 点击"确定"保存修改

**注意事项**：
- 只有"待回复"状态的工单可以编辑
- 已回复的工单不能再修改
- 如果工单已被回复，但您还有新问题，建议创建新的工单

#### 删除工单

**操作步骤**：
1. 在工单列表中，点击要删除的工单的"删除"按钮
2. 系统会弹出确认对话框
3. 点击"确定"确认删除

**注意事项**：
- 删除操作不可恢复，请谨慎操作
- 建议在问题解决后再删除工单，保留记录

### 6. 我的项目

**位置**：左侧菜单 → "我的项目"

**功能说明**：
在这里您可以查看所有与您相关的项目信息，包括项目详情、文件、链接和任务进度。

#### 查看项目列表

项目列表显示所有您参与的项目，包括：

- **项目名称**：项目名称
- **描述**：项目简要描述
- **完成度**：项目完成百分比
- **创建时间**：项目创建时间

#### 查看项目详情

**操作步骤**：
1. 在项目列表中，点击"查看详情"按钮
2. 系统会展开项目详情面板，包含以下标签页：

##### 项目概述

显示项目的详细概述，以 Markdown 格式展示，可能包括：
- 项目背景
- 项目目标
- 项目范围
- 重要说明

##### 项目文件

显示项目相关的所有文件，包括：

- **文件名**：文件名称
- **文件大小**：文件大小（自动转换为 KB 或 MB）
- **标签**：文件分类标签
- **上传时间**：文件上传时间

**文件筛选**：
- 在"按文件名筛选"输入框中输入文件名，可以快速找到文件
- 在"按标签筛选"输入框中输入标签，可以查看特定类型的文件

**下载文件**：
1. 找到要下载的文件
2. 点击"下载"按钮
3. 浏览器会自动开始下载文件

**注意事项**：
- 下载链接有时效性（通常1小时），请及时下载
- 如果下载失败，请刷新页面后重试

##### 项目链接

显示项目相关的所有链接，包括：

- **链接名称**：链接的标题
- **URL**：链接地址（可点击直接打开）
- **描述**：链接说明
- **创建时间**：链接创建时间

**使用建议**：
- 点击链接可以在新标签页中打开
- 建议收藏重要链接，方便后续访问

##### 进度管理

显示项目的任务列表和完成情况，包括：

- **完成度**：显示项目整体完成百分比
- **任务列表**：
  - **任务标题**：任务名称
  - **详情**：任务描述
  - **状态**：
    - **未完成**（灰色标签）：任务尚未完成
    - **已完成**（绿色标签）：任务已完成
  - **创建时间**：任务创建时间

**使用建议**：
- 定期查看任务列表，了解项目进度
- 关注未完成的任务，了解下一步工作

## 常见问题

### 登录相关

**Q: 忘记密码怎么办？**  
A: 请联系系统管理员重置密码。出于安全考虑，系统暂不支持自助找回密码。

**Q: 登录后提示"登录已过期"？**  
A: 系统登录状态会保持30分钟。如果长时间未操作，需要重新登录。

**Q: 可以同时在多台设备登录吗？**  
A: 可以，系统支持多设备同时登录。

### 工单相关

**Q: 工单提交后多久会回复？**  
A: 我们会在工作时间内尽快回复，通常在24小时内。紧急问题请通过电话联系。

**Q: 工单可以删除吗？**  
A: 可以，但删除后无法恢复。建议在问题解决后再删除，保留记录。

**Q: 已回复的工单可以继续回复吗？**  
A: 不可以。如果还有新问题，请创建新的工单。

**Q: 工单可以修改吗？**  
A: 只有"待回复"状态的工单可以修改。已回复的工单不能再修改。

### 业务账号相关

**Q: 业务账号申请后多久能审核通过？**  
A: 通常在1-3个工作日内完成审核。审核通过后，管理员会为您分配专属邮箱。

**Q: 专属邮箱和用户识别码可以修改吗？**  
A: 不可以，这些由管理员分配。如需修改，请联系管理员。

**Q: 业务资料填写不完整会影响使用吗？**  
A: 不会影响系统使用，但建议填写完整，以便顺利开具发票和邮寄文件。

### 合同相关

**Q: 可以修改合同内容吗？**  
A: 不可以，合同内容为只读。如需修改，请联系管理员。

**Q: 合同在哪里查看？**  
A: 在左侧菜单"我的合同"中查看。

**Q: 如何更新合同的签署状态？**  
A: 在"我的合同"页面的合同列表中，使用签署状态开关即可切换状态。开关显示"待签署"表示未签署，"已签署"表示已完成签署。

**Q: 签署状态可以随时更改吗？**  
A: 可以。您可以根据实际情况随时更新签署状态，系统会立即保存更改。

**Q: 签署状态更改后，管理员能看到吗？**  
A: 可以。状态更改后，管理员在管理后台可以看到最新的签署状态。

### 项目相关

**Q: 可以上传项目文件吗？**  
A: 不可以，文件由管理员上传。如需上传文件，请联系管理员或通过工单系统提交。

**Q: 文件下载失败怎么办？**  
A: 下载链接有时效性，请刷新页面后重新获取下载链接。如果问题持续，请通过工单系统反馈。

**Q: 项目完成度是如何计算的？**  
A: 完成度根据项目任务的完成情况自动计算。已完成任务数 / 总任务数 × 100%。

### 其他问题

**Q: 系统支持手机访问吗？**  
A: 支持。系统响应式设计，可以在手机浏览器中正常使用。

**Q: 数据安全吗？**  
A: 是的。所有数据都经过加密保护，只有您本人可以查看自己的信息。我们严格遵守数据保护法规。

**Q: 如何联系技术支持？**  
A: 您可以通过以下方式联系我们：
- 在系统中提交工单
- 发送邮件到技术支持邮箱
- 拨打客服电话
