# 客户帮助-注册Git服务器账户

> H01 注册Git服务器账户

本文档将会引导新用户在云梦镜像代码托管仓库中注册账号并使用代码仓

::: warning
注意，在您注册代码仓库账户时，需要确保您在[云梦镜像CSA](https://platform.dreamreflex.com)中已经注册了一个账户，由于代码仓服务依赖于统一身份认证源，因此您需要使用CSA中的账户进行注册。
:::

## 1. 点击登录按钮

![image-20260118204312565](./H01-注册Git服务器账户.assets/image-20260118204312565.png)

## 2. 点击使用云梦镜像CSA登录

![image-20260118204336820](./H01-注册Git服务器账户.assets/image-20260118204336820.png)

## 3. 登录成功后点击同意授权

![image-20260118204422855](./H01-注册Git服务器账户.assets/image-20260118204422855.png)

## 4. 点击注册账号

![image-20260118204438341](./H01-注册Git服务器账户.assets/image-20260118204438341.png)

其他内容保留默认即可

![image-20260118204452454](./H01-注册Git服务器账户.assets/image-20260118204452454.png)

您在此填写的任何内容都不会对项目产生任何影响，但您需要告知管理员您的注册用户名或邮箱，通常这和您在CSA中的信息保持一致

## 5. 完成注册

![image-20260118204729394](./H01-注册Git服务器账户.assets/image-20260118204729394.png)

在您进入到此页面后说明注册完成

## 6. 登录

当您的凭据过期，更换设备，更换浏览器时，只需要再次点击登录

![image-20260118204827763](./H01-注册Git服务器账户.assets/image-20260118204827763.png)

点击云梦镜像CSA登录后，点击同意，即可直接登录到服务器中

![image-20260118204852619](./H01-注册Git服务器账户.assets/image-20260118204852619.png)

此过程不会要求您输入密码，您也可以在Git服务器中创建您的仓库，但您在此仓库创建的内容云梦镜像拥有最终所有权。

## 7. 用户须知

通常您在注册项目后，我们将会让您的账户加入到项目组织，项目组织通常由多个仓库组成，这些仓库的版权，开放源代码决定权，开放源代码许可证和作者信息通常由用户指定，云梦镜像在未经客户许可或没有额外说明的情况下不会干预您对代码的处理意见。

您的相关信息默认是保密的，通常我们不会审查您的仓库，自动化程序也会通过脱敏匿名化的方式来帮助我们优化服务，但您的代码，相关产物，衍生物和再发行版本如果因您自己的公开发行和传播导致触犯相关法律法规，我们有权依法配合执法机关提供您的数据，包括不限于导出审计日志，导出数据库信息，用户信息，提交历史，提交公钥和LFS文件。
