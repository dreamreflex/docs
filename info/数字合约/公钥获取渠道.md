### 4. TXT 记录

我们利用DNS体系的TXT记录记录了我们的根证书信息

存储根证书的域名为

```
ca.dreamreflex.com
```

您可以通过下列脚本获取文件

如果您是Linux系统:

```bash
dig +short TXT ca.dreamreflex.com | tr -d '"' | tr -d '\n' | \
awk '{print "-----BEGIN CERTIFICATE-----\n"$0"\n-----END CERTIFICATE-----"}' > ca.pem
```

或

```bash

host -t TXT ca.dreamreflex.com | sed 's/.*TXT "\(.*\)"/\1/' | tr -d '\n' | \
awk '{print "-----BEGIN CERTIFICATE-----\n"$0"\n-----END CERTIFICATE-----"}' > ca.pem
```

如果您是Windows系统（支持PowerShell）:

```powershell
$txt = (Resolve-DnsName -Type TXT ca.dreamreflex.com).Strings -join ""
$pem = "-----BEGIN CERTIFICATE-----`n$txt`n-----END CERTIFICATE-----"
Set-Content -Path ca.pem -Value $pem
```

### 5. CERT记录

CERT是 RFC 4398 定义的一种 专门用来存储证书、公钥等安全材料的 DNS 记录类型。我们目前无法支持CERT记录来进行认证。
